## 🔍 Troubleshooting common errors
If you still see errors, check these common points of failure.

### Software reports a missing driver
If VueScan shows a "Scanner not found" message, Windows might use a generic driver that blocks the software.
*   Open the "Device Manager" in Windows.
*   Find your scanner in the list.
*   Right-click the device and select "Update driver".
*   Choose "Search automatically for drivers".
*   Restart your computer.

### The installation window freezes
Sometimes, antivirus software blocks the installation of hardware drivers.
*   Pause your antivirus software for 10 minutes.
*   Run the VueScan installer again.
*   Turn your antivirus back on after the installation finishes.

### Permission issues
Windows 11 features strict security settings. If the installer stops, verify your user account has administrative rights. You can check this in the "User Accounts" section of the Control Panel. If you use a Standard account, log into an Administrator account before you try the setup again.

## 💻 Tips for success
*   **Use USB ports directly on the PC:** If you use a desktop, plug the scanner into a USB port on the back of the case. Front ports sometimes lack the power needed for scanner hardware.
*   **Check cables:** A faulty USB cable causes many installation failures. Use the original cable if possible.
*   **Clean old software:** If you tried to install an older version before, use the "Add or remove programs" tool in Windows to fully delete the old files before you run the new installer.

## 📝 Frequently asked questions

**Do I need a paid license to complete the installation?**
You can install the trial version without a license. A watermark will appear on your scans until you enter a valid serial number.

**Can I move the installed files to another drive?**
No. The installer writes records to the Windows Registry. You must use the installer to move the program.

**Where does VueScan store my settings?**
The software stores settings in an initialization file within the installation directory. Keep this file if you want to back up your custom preferences.
